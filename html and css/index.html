A* Algorithm

Simple

The A* algorithm is a pathfinding algorithm that efficiently finds the shortest path between two points in a graph by considering both the cost to reach each node and an estimate of the remaining distance to the goal.

Genetic algorithm

Simple

Genetic algorithm is a computational method inspired by biological evolution, where solutions to optimization problems evolve through generations of selection, crossover, and mutation, mimicking natural selection to find optimal or near-optimal solutions.

Ant algorithm

Simple

Ant algorithm is a metaheuristic inspired by the foraging behavior of ants, where artificial ants iteratively build solutions by depositing pheromone trails on edges, favoring paths with higher pheromone levels, thereby finding optimal paths in a graph or network.

Decision Tree

Simple

Decision Tree is a predictive model that recursively splits data based on features to make decisions, forming a tree-like structure where each internal node represents a feature, and each leaf node represents a class label or value.
